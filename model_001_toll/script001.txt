### Description

Simple toll model.

Cars arrive at a 30-second interval (exponential distribution).

Cars are serviced at an interval of 20 seconds (exponential distribution).

The simulation runs for 10 hours and serves 1208 cars.

### List of acronyms (standard)

MB = Model Builder
OS = Object Selector
IE = Input Editor
KI = Key Inputs
OV = Output Viewer
* = modify or show

### Configure simmulation

OS / Simulation / IE - Simulation / KI
* Value RunDuration : 10 h

### Model building

MB-OS / Process Flow / SimEntity
* name SimEntity1 (F2) : Car

MB-OS / Process Flow / EntityGenerator
* name EntityGenerator1 (F2) : ArrivalCars

IE - ArrivalCars / KI
* Value PrototypeEntity : Car

MB-OS / Process Flow / Queue
* name Queue1 (F2) : CarsQueue

IE - ArrivalCars / KI
* Value NextComponent : CarsQueue

MB-OS / Process Flow / Server
* name Server1 (F2) : Tool

MB-OS / Process Flow / EntitySink
* name EntitySink1 (F2) : CarsOutput

IE - Toll / KI
* Value WaitQueue : CarsQueue
* Value NextComponent : CarsOutput

MB-OS / Probability Distributions / ExponentialDestribution
* name ExponentialDestribution1 (F2) : ArrivalFee

MB-OS / Probability Distributions / ExponentialDestribution
* name ExponentialDestribution1 (F2) : AttendanceFee

IE - ArrivalFee / KI
* Value UnitType : TimeUnit
* Value Mean: 30 s

IE - AttendanceFee / KI
* Value UnitType : TimeUnit
* Value Mean: 20 s

IE - ArrivalCars / KI
* Value InterArrivalTime : ArrivalFee

IE - Toll / KI
* Value ServiceTime : AttendanceFee

### Result after running

OV - CarsOutput
* LinkedComponent / NumberProcessed = 1208
