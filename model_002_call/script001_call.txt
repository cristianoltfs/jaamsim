### Description

Modelo de atendimento de ligação.

A ligação pode ser rejeitada, caso a capacidade esteja cheia.

A ligação pode ser abandonada, caso um período de espera seja alcançado.

Parâmetros:

Taxa de chegada ligação = 1.5 por segundo (1 ligação há cada 0.67 segundos).

Tempo de abandono da ligação = 100 segundos.

Número de Operators = 100.

Tempo de atendimento = Triangular(100/2, 100, 2*100).

Capacidade máxima da fila = 50.

### List of acronyms (standard)

MB = Model Builder
OS = Object Selector
IE = Input Editor
KI = Key Inputs
OV = Output Viewer
* = modify Value/Name or show
